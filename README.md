# 中国反制裁法律体系网站

一个面向律师、合规和法务的静态研究站，收录中国反制裁法律体系及若干相关法域文本，并附来源链接、专题比较、时间线和分析文章索引。

## 本地预览

在项目根目录运行：

```powershell
python -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 内容扩展

- 法律文本数据：`src/data/laws.js`
- 分析文章数据：`src/data/articles.js`
- 专题比较：`src/data/topics.js`
- 时间线：`src/data/timeline.js`

## 维护原则

- 中文法律文本优先使用官方公开来源。
- 英文文本必须区分官方英文与非官方英文。
- 分析文章默认仅提供摘要与链接；确认可全文收录时再补全文。
- 任何来源不清晰的文本，都应在站内明确标示，而不要伪装为正式版本。

## 版权提示

法律原文优先使用公开官方来源。分析文章遵循保守处理原则：除已确认可合法转载的材料外，站内仅保留摘要、出处与原链接，避免超出版权边界.
